## 1. 사진·영상 정리: 가장 빠르게 용량 확보하는 방법

스마트폰 저장공간의 **40~60%를 차지하는 주범은 사진과 영상**입니다. 특히 고해상도 영상이나 RAW 포맷 사진은 파일 하나당 수십 MB를 차지합니다.

### 효과적인 사진 정리 순서

1. **중복·유사 사진 삭제**: 같은 장소에서 연속으로 찍은 사진들을 모아서 제일 잘 나온 한 장만 남기기
2. **스크린샷 일괄 삭제**: 스크린샷 폴더는 대부분 확인 후 삭제 가능
3. **WhatsApp·카카오톡 미디어 정리**: 메신저에서 받은 영상은 별도 폴더에 자동 저장됨

### 클라우드로 이동 후 원본 삭제

| 서비스 | 무료 용량 | 특징 |
|--------|---------|------|
| 구글 포토 | 15GB | 안드로이드 기본 연동 |
| iCloud | 5GB | 아이폰 기본 연동 |
| 네이버 MYBOX | 30GB | 국내 사용자에 유리 |
| 원드라이브 | 5GB (M365 구독 시 1TB) | Office 연동 강점 |

> **꿀팁**: 구글 포토·iCloud에 업로드 완료 후 '기기에서 삭제' 옵션을 사용하면 클라우드에는 남기고 폰 용량만 확보할 수 있습니다.

## 2. 앱 캐시·데이터 삭제

앱을 오래 사용하면 캐시(임시 파일)가 수백 MB~수 GB씩 쌓입니다. **앱을 삭제하지 않고도 캐시만 지우면** 용량을 크게 확보할 수 있습니다.

  <figcaption>▲ 앱 캐시를 정기적으로 삭제하면 저장공간과 속도 모두 개선됩니다 / AI 생성 이미지</figcaption>
</figure>

### 갤럭시(안드로이드) 캐시 삭제 방법

```
설정 → 앱 → 앱 선택 → 저장공간 → 캐시 삭제
```

캐시가 많이 쌓이는 앱 순위:
- 유튜브 (오프라인 영상·썸네일)
- 카카오톡 (사진·동영상 캐시)
- 인스타그램 / 틱톡 (피드 이미지 캐시)
- 네이버·다음 지도 (오프라인 지도 데이터)
- 넷플릭스 (다운로드 영상)

### 아이폰 캐시 삭제 방법

아이폰은 앱별 캐시를 직접 삭제하는 기능이 없습니다. 대신 아래 방법을 사용하세요:

1. **앱 오프로드(Offload)**: `설정 → 일반 → iPhone 저장 공간 → 앱 오프로드` — 앱 데이터는 유지하고 앱 본체만 삭제, 나중에 재설치 시 데이터 복원
2. **앱 완전 삭제 후 재설치**: 캐시까지 완전 초기화
3. **Safari 캐시**: `설정 → Safari → 방문 기록 및 웹 사이트 데이터 지우기`

## 3. 잘 안 쓰는 앱 삭제 & OTT 다운로드 콘텐츠 정리

설치 후 한 번도 안 쓴 앱, 한 번 보고 저장해둔 OTT 영상이 의외로 큰 용량을 차지합니다.

### 앱 사용 빈도 확인하기

- **갤럭시**: 설정 → 디지털 웰빙 → 앱 사용 시간 확인
- **아이폰**: 설정 → 스크린 타임 → 앱 사용 내역

3개월 이상 사용하지 않은 앱은 과감히 삭제하세요.

### OTT 다운로드 영상 삭제

| OTT | 다운로드 관리 경로 |
|-----|----------------|
| 넷플릭스 | 내 다운로드 → 개별 삭제 또는 전체 삭제 |
| 유튜브 프리미엄 | 라이브러리 → 다운로드 → 삭제 |
| 왓챠 / 웨이브 | 다운로드 메뉴 → 삭제 |

## 4. 메신저 미디어 파일 일괄 정리

카카오톡·텔레그램 같은 메신저는 받은 사진·영상을 자동 저장합니다. 이 파일들은 쌓이면 1~3GB 이상 되는 경우도 많습니다.

<figure>
  <img src="/assets/images/posts/2026-07-02-smartphone-storage-tips-3.jpg" alt="스마트폰 저장공간 정리 완료" loading="lazy">
  <figcaption>▲ 꾸준한 정리 습관으로 스마트폰을 쾌적하게 유지하세요 / AI 생성 이미지</figcaption>
</figure>

### 카카오톡 미디어 정리

```
카카오톡 → 설정 → 개인/보안 → 저장 공간 관리
```
- 대화별 미디어 파일 크기 확인 후 일괄 삭제 가능
- '미디어 자동 저장' 옵션을 OFF로 바꾸면 이후 파일이 쌓이는 것을 방지

### 텔레그램 미디어 정리

```
텔레그램 → 설정 → 데이터 및 저장소 → 저장공간 사용량
```

## 5. 시스템 파일·기타 용량 정리

갤럭시는 '기타'나 '시스템 데이터'로 분류된 파일들이 은근히 많은 용량을 차지하기도 합니다.

### 갤럭시 추천 정리 앱

- **갤럭시 디바이스 케어**: 기본 내장, 설정 → 배터리 및 디바이스 케어 → 저장공간
- **파일스 바이 구글(Files by Google)**: 무료, 중복 파일·잔여 파일 자동 감지 후 삭제 추천

### 아이폰 저장 공간 분석

```
설정 → 일반 → iPhone 저장 공간
```
각 앱별 사용 용량과 '권장 사항'을 확인할 수 있습니다. iCloud 백업, iMessage 첨부파일 등도 여기서 관리합니다.

## 핵심 요약 – 저장공간 확보 우선순위

| 우선순위 | 방법 | 예상 확보 용량 |
|---------|------|--------------|
| ★★★ | 사진·영상 클라우드 이동 후 삭제 | 5~30GB |
| ★★★ | OTT 다운로드 영상 삭제 | 1~10GB |
| ★★☆ | 메신저 미디어 파일 정리 | 1~5GB |
| ★★☆ | 앱 캐시 삭제 | 500MB~3GB |
| ★☆☆ | 잘 안 쓰는 앱 삭제 | 수백MB~수GB |

저장공간 정리는 한 번에 끝내는 것보다 **월 1회 습관적으로 점검**하는 게 훨씬 효과적입니다. 위 방법을 순서대로 실천하면 대부분의 스마트폰에서 **3~20GB 이상** 확보할 수 있습니다.
