## 1. TOTEM Ink Pad – 손바닥 위의 E잉크 메모 패드

손바닥 크기의 E잉크 디스플레이에 전자펜으로 메모를 남기는 기기입니다.  
스마트폰 알림 없이 **오직 메모·스케치만** 집중하고 싶은 사람을 위한 제품입니다.

| 항목 | 사양 |
|------|------|
| 디스플레이 | 5.8인치 E잉크, 1448×1072 (300 PPI) |
| 크기 | 145 × 100 × 6.5 mm |
| 무게 | 168 g |
| 배터리 | 2,000mAh (약 3주 사용) |
| 충전 | USB-C (15W PD) |
| 저장 | 32GB 내장 |
| 연동 | BLE 5.0 / 모바일 앱 동기화 |
| 가격 | 얼리버드 $129 (약 18만 원) |

**이런 분께 추천:** 회의 중 딴 생각 없이 메모만 하고 싶은 직장인, 스케치 연습생.  
**단점:** PDF 파일 가져오기 기능이 없고, 자체 앱 완성도가 베타 수준.

## 2. MOFT Snap Ring Pro – 자석 흡착 스마트링 거치대

맥세이프 호환 자석으로 스마트폰 뒤에 붙었다 떼어지는 **스마트링 + 킥스탠드 + 지갑** 3-in-1 기기입니다. 이미 MOFT 브랜드가 유명하지만, 이번 모델은 **360° 회전 힌지**와 **카드 2장 수납**을 추가한 업그레이드판입니다.

| 항목 | 사양 |
|------|------|
| 호환 | MagSafe(아이폰 15~17), 맥세이프 호환 안드로이드 |
| 소재 | 비건 레더 + 알루미늄 힌지 |
| 크기 (접힘) | 68 × 46 × 6 mm |
| 무게 | 38 g |
| 카드 수납 | 최대 2장 |
| 충전 | 없음 (별도 충전 불필요) |
| 가격 | 얼리버드 $39 (약 5.5만 원) |

**이런 분께 추천:** 지갑·케이스·거치대를 따로 들고 다니기 싫은 미니멀리스트.  
**단점:** 안드로이드는 맥세이프 호환 케이스 별도 구매 필요.

## 3. Auracle Clip – 옷깃에 클립하는 AI 음성 메모 기기

목에 걸거나 옷깃에 클립하면, 대화·강의·회의 내용을 자동으로 **AI 요약 텍스트로 변환**해주는 웨어러블 레코더입니다.  
Humane AI Pin의 실패를 교훈 삼아 **디스플레이를 완전히 제거**, 오직 녹음·AI 요약에 집중한 것이 특징입니다.

| 항목 | 사양 |
|------|------|
| 크기 | 52 × 28 × 11 mm |
| 무게 | 22 g |
| 마이크 | 듀얼 빔포밍 MEMS 마이크 |
| 배터리 | 400mAh (연속 녹음 8시간) |
| 충전 | 포고핀 (전용 케이스 충전, 케이스 용량 2,000mAh) |
| 저장 | 64GB 내장 |
| AI 요약 | 앱 연동 (iOS/Android), 한국어 지원 예정 |
| 가격 | 얼리버드 $89 (약 12.5만 원) |

> ⚠ **주의:** 출시일은 2026년 4분기 예정이며, 한국어 AI 요약 지원 시점은 미정입니다.

**이런 분께 추천:** 강의·인터뷰·회의 내용을 자동 정리하고 싶은 기자·학생·프리랜서.  
**단점:** 포고핀 충전 방식으로 범용성 낮음, 서버 구독 요금 월 $4.9 예정.

## 4. ZeroKey Pebble – 엄지손가락만으로 조작하는 1버튼 키보드

엄지손가락 하나로 모스 부호 방식의 입력을 학습해, 스마트폰 없이도 **스마트워치·AR글래스·스마트TV**를 제어하는 초소형 입력 장치입니다.  
학습 곡선이 있지만, 한 손이 자유롭지 않은 상황(운전 중 스마트워치 조작, 요리 중 레시피 스크롤 등)에 유용합니다.

| 항목 | 사양 |
|------|------|
| 크기 | 32 × 20 × 10 mm (엄지 링 타입) |
| 무게 | 9 g |
| 연결 | BLE 5.2 |
| 배터리 | 90mAh (약 2주 대기, 4시간 연속 사용) |
| 충전 | USB-C (5W) |
| 호환 | iOS, Android, Windows, macOS, WearOS |
| 가격 | 얼리버드 $59 (약 8.3만 원) |

**이런 분께 추천:** AR/VR 기기 사용자, 스마트홈 무선 리모컨 대체품을 찾는 분.  
**단점:** 모스 부호 학습에 평균 2~3주 소요, 실용화까지 인내심 필요.

## 구매 전 꼭 확인할 것

크라우드펀딩 제품은 **정식 출시 제품이 아닙니다.** 아래 사항을 반드시 점검하세요.

1. **펀딩 달성률·후원자 수** 확인 – 최소 200% 이상, 후원자 1,000명 이상이면 비교적 안전
2. **제조사 이전 출시 이력** 확인 – 처음 등장하는 브랜드는 리스크 높음
3. **배송 예상일** 현실적으로 판단 – 평균 3~6개월 지연 일반적
4. **환불 정책** 사전 확인 – 킥스타터는 원칙적으로 환불 불가
5. **AS·보증 정책** 확인 – 소규모 업체는 AS 어려울 수 있음
