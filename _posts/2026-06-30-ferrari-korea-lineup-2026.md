이탈리아 마라넬로에서 탄생한 **페라리(Ferrari)**는 단순한 스포츠카 브랜드를 넘어 '드라이빙의 철학'을 상징합니다. 2026년 현재 한국 페라리 공식 딜러(FMK)를 통해 구매할 수 있는 주력 모델은 5종입니다. V6 하이브리드부터 V12 자연흡기 SUV까지, 각 모델의 제원·가격·특징을 낱낱이 정리했습니다.

---

## 1. Ferrari 296 GTB – V6 하이브리드 미드십 스포츠카

<figure class="car-figure">
  <img src="https://upload.wikimedia.org/wikipedia/commons/thumb/4/4e/Ferrari_296_GTB_Assetto_Fiorano_rear.png/1280px-Ferrari_296_GTB_Assetto_Fiorano_rear.png" alt="페라리 296 GTB 앗세토 피오라노 리어" loading="lazy">
  <figcaption>▲ Ferrari 296 GTB Assetto Fiorano / 출처: Wikimedia Commons (CC BY 2.0)</figcaption>
</figure>

296 GTB는 페라리가 V6 엔진을 미드십에 얹은 첫 로드카입니다. 전기모터와의 결합으로 **830ps**라는 수치를 내면서도 차체 무게는 1,470kg에 불과합니다. 트랙 특화 사양인 '앗세토 피오라노' 패키지를 선택하면 타이탄 배기·카본 에어로가 추가됩니다.

| 항목 | 수치 |
|------|------|
| 엔진 | 2,992cc V6 120° 트윈터보 + 전기모터 |
| 최고출력 | 830ps (610kW) @ 8,000rpm |
| 최대토크 | 740Nm (75.5kgf·m) |
| 0→100km/h | **2.9초** |
| 0→200km/h | 7.3초 |
| 최고속도 | 330km/h 이상 |
| 전장 × 전폭 × 전고 | 4,565 × 1,958 × 1,187mm |
| 축거 | 2,600mm |
| 공차중량 | 1,470kg |
| 연비 (WLTP 복합) | 약 6.5km/L |

**한국 판매 가격:** 약 **3억 5,000만 원~** (트림·옵션에 따라 상이)

## 2. Ferrari SF90 스트라달레 – 1,000마력 PHEV 하이퍼카

<figure class="car-figure">
  <img src="https://upload.wikimedia.org/wikipedia/commons/thumb/d/d4/Ferrari_SF90_Stradale_4.jpg/1280px-Ferrari_SF90_Stradale_4.jpg" alt="페라리 SF90 스트라달레" loading="lazy">
  <figcaption>▲ Ferrari SF90 Stradale / 출처: Wikimedia Commons (CC BY-SA 4.0)</figcaption>
</figure>

SF90은 페라리 양산차 최초로 시스템 출력 **1,000ps**를 돌파한 플러그인 하이브리드(PHEV)입니다. V8 엔진에 전기모터 3개를 결합해 0→100km/h를 **2.5초**에 주파하며, 순수 전기 모드(eManettino)로 시내에서 조용히 25km를 달릴 수도 있습니다.

| 항목 | 수치 |
|------|------|
| 엔진 | 3,990cc V8 트윈터보 + 전기모터 3개 (PHEV) |
| 시스템 최고출력 | 1,000ps (735kW) |
| 0→100km/h | **2.5초** |
| 0→200km/h | 6.7초 |
| 최고속도 | 340km/h |
| 전기 주행거리 | 25km (WLTP) |
| 전장 × 전폭 × 전고 | 4,710 × 1,972 × 1,186mm |
| 축거 | 2,650mm |
| 공차중량 | 1,570kg |

**한국 판매 가격:** 약 **6억 원~** (Spider·Assetto Fiorano 패키지 별도)

## 4. Ferrari 푸로산구에 – V12 자연흡기 슈퍼 SUV

<figure class="car-figure">
  <img src="https://upload.wikimedia.org/wikipedia/commons/thumb/6/6d/2023_Ferrari_Purosangue.jpg/1280px-2023_Ferrari_Purosangue.jpg" alt="페라리 푸로산구에" loading="lazy">
  <figcaption>▲ Ferrari Purosangue (2023) / 출처: Wikimedia Commons (CC BY-SA 4.0)</figcaption>
</figure>

푸로산구에(Purosangue, 이탈리아어로 '순혈')는 페라리 역사상 최초의 4도어 SUV입니다. 그러나 일반 SUV와 달리 **V12 6.5L 자연흡기** 엔진을 앞뒤 차축 중앙에 탑재하고, 뒷좌석까지 버터플라이 도어로 개방합니다. 전자식 능동형 서스펜션으로 스포츠카에 가까운 코너링을 구현합니다.

| 항목 | 수치 |
|------|------|
| 엔진 | 6,496cc V12 자연흡기 |
| 최고출력 | 725ps (533kW) @ 7,750rpm |
| 최대토크 | 716Nm (73.0kgf·m) @ 6,250rpm |
| 0→100km/h | **3.3초** |
| 최고속도 | 310km/h |
| 전장 × 전폭 × 전고 | 4,973 × 2,028 × 1,589mm |
| 축거 | 3,018mm |
| 공차중량 | 2,033kg |
| 연비 (WLTP 복합) | 약 4.5km/L |

**한국 판매 가격:** 약 **5억~6억 원~**

## 5. Ferrari 12Cilindri – V12의 귀환

2024년 공개된 12Cilindri는 페라리의 V12 엔진을 프론트 미드십에 얹은 순수 내연기관 그란투리스모입니다. 830ps @ 9,000rpm이라는 고회전 V12 특유의 사운드가 최대 강점입니다. 쿠페와 Spider 두 가지 바디타입이 있습니다.

| 항목 | 수치 |
|------|------|
| 엔진 | 6,496cc V12 자연흡기 |
| 최고출력 | 830ps (610kW) @ 9,000rpm |
| 최대토크 | 678Nm (69.1kgf·m) @ 7,250rpm |
| 0→100km/h | **2.9초** |
| 최고속도 | 340km/h 이상 |
| 공차중량 | 1,560kg (쿠페) |
| 연비 (WLTP 복합) | 약 5.8km/L |

**한국 판매 가격:** 약 **5억~6억 원~**

## 마치며

페라리는 '가장 빠른 차'를 만드는 것이 아니라 '가장 강렬한 드라이빙 경험'을 설계하는 브랜드입니다. 296 GTB가 미래 지향적 하이브리드 기술을 담았다면, 12Cilindri는 V12의 감성을 수호합니다. 한국에서는 **FMK(페라리 모터스 코리아)**를 통해 구매 상담이 가능하며, 공식 딜러는 서울 강남·청담, 부산 등에 위치합니다.

> **구매 전 주의 사항:** 페라리는 공장 주문 제작 방식으로 대기 기간이 6개월~2년 이상 소요될 수 있으며, 실제 가격은 옵션·색상·패키지에 따라 크게 달라집니다.
