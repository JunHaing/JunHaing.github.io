구글 드라이브는 무료로 15GB를 제공하는 클라우드 저장 서비스입니다. 많은 분들이 단순히 파일 저장 용도로만 쓰시지만, 숨겨진 기능을 알면 업무·학습 생산성이 크게 올라갑니다. 2026년 기준 꼭 알아야 할 활용법 10가지를 정리했습니다.

---

## 1. 무료 15GB, 효율적으로 쓰는 법

구글 계정 하나당 Gmail·드라이브·포토가 **15GB를 공유**합니다. 저장 공간이 부족하다면 아래 순서로 정리하세요.

| 확인 순서 | 방법 |
|---------|------|
| ① 큰 파일 확인 | 드라이브 → 저장용량 → 저장용량 절약 |
| ② Gmail 정리 | 첨부파일 큰 메일 검색: `has:attachment larger:10m` |
| ③ 구글 포토 | 화질을 '저장용량 절약 화질'로 변환 |
| ④ 공유 드라이브 | 다른 사람이 공유한 파일은 내 용량 미차감 |

> **팁**: 구글 계정을 2개 만들면 사실상 30GB를 무료로 활용할 수 있습니다.

## 2. 폴더 구조 & 색상 지정으로 빠른 탐색

파일이 많아질수록 폴더 구조가 핵심입니다.

- **폴더 색상 변경**: 폴더 우클릭 → 색상 변경 → 한눈에 구분
- **별표(★) 표시**: 자주 쓰는 파일·폴더에 별표 → 왼쪽 메뉴 '별표 표시됨'에서 바로 접근
- **단축 드라이브**: 드라이브 내 어디서든 파일 '바로가기' 생성 (원본 이동 없이)

**추천 폴더 구조 예시**
```
📁 업무
  📁 2026년
    📁 1분기
    📁 2분기
📁 개인
  📁 문서
  📁 사진백업
📁 공유받은자료
```

## 3. 공유 설정 완전 정복

구글 드라이브 공유는 3가지 권한으로 나뉩니다.

| 권한 | 설명 | 언제 사용? |
|------|------|-----------|
| 뷰어 | 보기만 가능 | 외부 공개 자료 |
| 댓글 작성자 | 댓글 추가 가능 | 피드백 요청 시 |
| 편집자 | 수정·삭제 가능 | 협업 팀원 |

**링크 공유 보안 설정**
- '링크가 있는 모든 사용자' → 외부 공유 시 주의
- '제한됨' → 지정된 사람만 접근 (기본 설정 권장)
- 공유 만료일 설정 가능 (기업용 구글 워크스페이스)

## 4. 오프라인 모드 – 인터넷 없이도 작업

와이파이 없는 환경에서도 구글 드라이브 파일을 편집할 수 있습니다.

**설정 방법**
1. 크롬 브라우저에서 drive.google.com 접속
2. 설정(⚙) → '오프라인' 항목 활성화
3. 크롬 확장프로그램 'Google Docs 오프라인' 설치

작업 후 인터넷 연결 시 자동으로 동기화됩니다.

| 단축키 | 기능 |
|--------|------|
| `N` | 새 파일/폴더 만들기 |

| `Shift + Z` | 여러 위치에 추가 |
| `/` | 검색창 포커스 |
| `Ctrl + Enter` | 파일 열기 |
| `Ctrl + Shift + C` | 공유 가능한 링크 복사 |

## 6. 구글 문서·스프레드시트 활용 팁

구글 드라이브 안의 **구글 문서(Docs)**, **스프레드시트(Sheets)**, **프레젠테이션(Slides)**은 마이크로소프트 오피스 대체재로 충분합니다.

- **버전 기록**: 파일 → 버전 기록 → 최대 30일 이전으로 복원 가능
- **음성 입력**: 도구 → 음성 입력 → 마이크로 직접 받아쓰기
- **탐색 기능**: 도구 → 탐색 → AI 기반 데이터 분석 및 요약
- **스마트 캔버스**: 문서 내 @멘션으로 파일·이벤트 삽입

---

## 7. 사진·영상 자동 백업 설정

스마트폰 사진을 구글 드라이브로 자동 백업하려면:

1. **구글 포토** 앱 설치 (드라이브와 별도지만 연동 가능)
2. 백업 설정 → '저장용량 절약 화질' 선택 (무제한 무료)
3. 드라이브 앱에서도 '카메라 업로드' 설정 가능

> **주의**: 2021년 이후 구글 포토 '저장용량 절약' 화질도 15GB에 포함됩니다.

---

## 8. 검색 고급 필터 사용법

파일이 많을 때는 검색 연산자를 활용하세요.

| 검색어 | 설명 |
|--------|------|
| `type:spreadsheet` | 스프레드시트만 검색 |
| `type:pdf` | PDF만 검색 |
| `owner:me` | 내가 소유한 파일 |
| `before:2025-01-01` | 특정 날짜 이전 파일 |
| `title:보고서` | 파일명에 '보고서' 포함 |

## 9. 구글 드라이브 API & 자동화

개발자나 파워 유저라면 **자동화 연동**을 고려해보세요.

- **구글 앱스 스크립트**: 드라이브 파일을 자동으로 정리·이름 변경·이동
- **Zapier / Make**: 다른 앱과 연결해 자동으로 파일 저장
- **Google Drive API**: Python·JavaScript로 파일 업로드·다운로드 자동화

예시: 이메일 첨부파일을 특정 드라이브 폴더에 자동 저장
